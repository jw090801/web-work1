F2 : 스크린샷
F3 : 좌표와 지형 등등 F3 + H : 아이템 코드 번호와 아이템의 내구도가 숫자로 표시됩니다.(내구도는 1이라도 소모되어야 나타납니다.)
F5 : 시점변경
F8 : 부드러운 시선이동
F11 : 전체창
tab : 멀티에서 서버에 접속한 사람들의 목록을 볼 수 있다.
shift : 웅크리기
shift + 방향키 : 떨어지지 않습니다.
조합창에서 shift 우클릭/좌클릭 : 조합가능한 최대 개수 한번에 조합 가능하다.
shift + 아이템 마우스 우클릭 : 상자 => 인벤토리 or 인벤토리 => 상자
아이템 우클릭 : 아이템이 반만 집어진다.
아이템을 클릭한 상태에서 우클릭 : 들고있는 아이템의 갯수가 한개씩 줄어든다.
아이템 더블클릭 : 인벤토리의 흩어져있는 같은 종류의 아이템들을 한번에 모아줌 (1.5.2 이상부터 가능)
Q : 들고있는 아이템 버리기
아이템 위에 마우스 두고 Q (클릭x) : 선택한 아이템 버리기
insert : 채팅을 키고 insert를 누르면 색깔의 코드번호를 알 수 있다.
(색깔코드를 입힌 채팅은 멀티에서만 되는 것 같네요/기능이 들어가있는 종류의 한글패치 적용상태에서 된다고 하네요.)
채팅으로 누군가 사이트의 링크를 올리면 클릭을 통해서 그 링크 사이트로 들어갈 수 있다. (찍힌 스크린샷이 있는 폴더로도 연결 가능하다.)
마우스 휠 말고도 1~9의 숫자 버튼을 통해 들고있는 아이템을 바꿀 수 있다.
/gamerule 이란 명령어로 여러가지 게임 내의 설정이 가능합니다. keepinventory 플러그인 없이도 죽을시의 아이템이 떨어지지 않게 할 수 있습니다.
+추가 ( 1.6부터는 안되는것 같네요 )
F3 + F : 단축키로 시야설정이 가능하다. (shift 누르면 가까이서부터 안누르면 멀리서부터)
F3 + P : 게임일시정지 여부 결정
F3을 눌러 좌표를 띄운 상태에서 1~9 : 오른쪽에 더 자세한 그래프를 띄운다.
F3을 눌러 좌표를 띄운 상태에서 0 : 오른쪽에 간단한 그래프를 띄운다.
F10 : 마인크래프트 멈춤
F3 + A : 청크 리로드
F3 + T : 텍스쳐팩 리로드
F3 + B : 히트박스가 흰색 선으로 표시된다.
F3 + S : 사운드 리로드
F3 + C : 크래쉬뜹니다. (6~10초간 누르고 있어야 한다네요)
아이템 좌클릭 + 드래그 : 해당 아이템이 동등하게 분배된다.
ctrl + Q : 한꺼번에 버리기
